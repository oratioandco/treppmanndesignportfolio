§ 11 · DRAFT V11 — diffusion-style resolution cycle. The scene refines from chunky blocks to crisp pixels then holds, like watching a generation pass complete. (CURRENT)

V11 · scene refines through resolution stages (32→16→8→4→2→1) then holds crisp for ~5s. Same v10 animated objects continue underneath STATE: SUNNY

Approach: every frame the full v10 scene (silhouettes + atmospheric depth + bird + smoke + sun + weather) is drawn to an offscreen canvas at native 480x180. Then it's downsampled to (W/N × H/N) via drawImage with imageSmoothingEnabled=false, then upsampled back to 480x180 — chunky pixel blocks of size N. Stages cycle: N=32 (~0.6s) → 16 → 8 → 4 → 2 → 1 (~5s hold) → loop. Roughly 7s per generation cycle. Same algorithm could trigger on Intersection Observer for one-shot scroll-reveal on cover images.

§ 10 · DRAFT V10 — Monkey Island style: static pixel-art silhouettes with atmospheric depth, smooth-animated pixel objects (bird, smoke, sun bob). No pixel flicker. (CURRENT)

V10 · static silhouettes + atmospheric depth + smooth-animated pixel objects (bird, smoke, sun-bob) STATE: SUNNY

Monkey Island principle: pixel art is the shape language; smooth animation is how the world breathes. Pixels never flicker — whole objects translate / fade / rotate. Atmospheric depth via opacity: distant buildings 50%, crane 65%, mid buildings 80%, Dom 90%, TV Tower 95%, foreground tree 100%. A pixel-sprite bird flies across with 2-frame wing flap. A pixel smoke plume rises from a distant chimney (puffs translate up + fade out over ~6s). Sun gently bobs Y±2px. Cream cross still fades into the TV Tower sphere when sunny.

§ 09 · DRAFT V9 — flowing 2D noise field. Glistening highlights move coherently across silhouettes, not random flicker. (CURRENT)

V9 · FBM noise field flows with time · coherent moving highlights, not random pixels STATE: SUNNY

Approach: hash-based 2D value noise + 2-octave FBM + time offset = a smooth animated noise field. Each render frame, sample the noise at every silhouette/edge cell position. Interior cells render TEAL by default but "punch through" to cream when noise > 0.62 (creates moving highlight patches). Edge cells render TEAL only when noise > 0.72 (creates boiling-out extensions where the noise field peaks). Same field drives both — so the highlight patches and the edge extensions are correlated, sweeping across the scene together.

§ 08 · DRAFT V8 — vanilla canvas cellular automaton on silhouette edges. Real CA, not 180 hand-placed dots. (CURRENT)

V8 · vanilla canvas (5kb) · CA steps ~10fps, render 60fps · silhouette edges genuinely breathe STATE: SUNNY

Rules now affect the ENTIRE shape, not just the outer crust. Two rule regimes per cell: Interior (inside silhouette): survival 0.94 + 0.005×n, revival 0.72 + 0.03×n. Mostly always alive but with ~5% steady-state dropouts that quickly refill. Edge (just outside silhouette): survival 0.50 + 0.10×n, birth 0.06 + 0.08×n. Boils outward. The silhouette is no longer drawn statically — the CA's alive set IS the rendered shape.

§ 03 · OPEN ISSUES & NEXT MOVES

v2 changes vs v1

  • Filled silhouettes → outlined / line-art via stroke-dasharray for the pixel-broken edge character
  • Three dash patterns in use: out-tight (3 1) for foreground, out-loose (2 2) for mid-ground, out-sparse (1 3) for distant buildings — gives each element its own texture, like Redaction's per-character distress
  • Reichstag dropped (was the weakest element)
  • TV Tower sphere redrawn as a discrete pixel circle (rect cluster forming a round shape)
  • Building rooflines now varied — stepped, slanted, flat — for skyline rhythm
  • Berliner Dom gets a sparse colonnade hint (vertical 1px ticks on the base)
  • Tree foliage = outlined circle with a few internal "leaf" dots, not a filled blob
  • Ground line is now dashed too — pixel-broken

If v2 is the direction

  • Add animation: stroke-dashoffset shifts subtly over time to give the outlines a "shimmer / alive" feel like the Redaction headline
  • Clouds drift slowly horizontally (60s loop)
  • Rain falls (~2s loop, multiple offsets)
  • Sun pulses gently (radius / opacity, 5s loop)
  • Pope's Revenge cross fades in over 1.5s when sunny state activates
  • Then wrap in BerlinFooter.astro + add Open-Meteo weather fetch + mount in Layout.astro
